# cockpit-accompagnement

## Sécurité Firebase (à appliquer)

Un modèle de règles RTDB renforcées est disponible ici :

- `data/config/realtime-database-rules.secure.json`

À copier/coller dans Firebase Console > Realtime Database > Règles, puis publier.

## Codes spéciaux (cockpit)

- `PROFPSE` est créé automatiquement (actif par défaut).
- `INVITE` est créé automatiquement (inactif par défaut).
- Tu peux ajouter des codes démo supplémentaires, les activer/suspendre/supprimer depuis le cockpit.
- Si un code est compromis: crée un nouveau code, puis suspend l'ancien.
